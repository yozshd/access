# GFM Extensions

These features come from [GitHub Flavored Markdown](https://github.github.com/gfm/) and require `gfm: true` in Astro's markdown config:

```js
// astro.config.mjs
markdown: {
  gfm: true,
},
```

Headings, lists, blockquotes, and code blocks are covered on [Base Markdown](/docs/notes/typography-examples).
