# YOZSH: Astro Basics Content List

## Project Structure

This Astro project is intended to list content.

```text
/
├── public/
├── src
│   ├── assets/
│   ├── components/
│   ├── content/
│   │   ├── data/
│   │   └── data/
│   ├── layouts/
│   ├── lib/
│   ├── pages/
│   └── styles/
├── astro.config.mjs
├── package.json
└── README.md
```

To learn more about the folder structure of an Astro project, refer to [Astro's guide on project structure](https://docs.astro.build/en/basics/project-structure/).
